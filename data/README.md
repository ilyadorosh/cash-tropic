# Data Directory

This directory contains runtime data files for the application.

## Files

- `saved_messages.json` - Stores user-saved chat messages (auto-generated, not committed to git)
- `cryonics.ts` - Configuration file for cryonics feature

## Notes

The `saved_messages.json` file is automatically created when users save their first message. It is excluded from version control via `.gitignore`.
