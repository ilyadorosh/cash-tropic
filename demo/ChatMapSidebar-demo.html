ChatMapSidebar Component

This sidebar component provides a visual navigation pane for browsing stored chats.

Features

  • Search: Filter chats by title or snippet text
  • Visual List: See all chats with titles, dates, and snippets
  • Sorting: Chats are ordered newest-first automatically
  • Navigation: Click any chat to open it with the snippet pre-filled
  • Hover Effects: Visual feedback on mouse hover
  • Keyboard Accessible: Full keyboard navigation support

Try It

Try hovering over the chat items in the sidebar to see the hover effects!

The search box filters chats in real-time (functional in this demo).
